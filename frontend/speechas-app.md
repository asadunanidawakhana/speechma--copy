# speechas-app

## Goal
Build "Speechas", a light-themed Text-to-Speech web application using Next.js (frontend) and FastAPI (backend) with `edge-tts` integration.

## Tasks
- [x] Task 1: Scaffolding → Verify: Both Next.js and FastAPI servers start successfully.
- [x] Task 2: Backend API Routes → Verify: `edge-tts` voice list and generation endpoints work.
- [x] Task 3: Backend Silence Removal → Verify: Audio processing correctly trims silence.
- [x] Task 4: UI Design System Setup → Verify: Tailwind and theme correctly configured per `ui-ux-pro-max`.
- [x] Task 5: Voice Catalog & Search → Verify: UI parses voice list and implements search functionality.
- [x] Task 6: Generation Interface & Player → Verify: User can type text, select options, play, and download audio.
- [x] Task 7: Final Integration → Verify: End-to-end flow works.

## Done When
- [ ] User can generate and download text-to-speech audio.
- [ ] User can browse and preview all `edge-tts` voices.
- [ ] Pitch and speed modifications apply correctly.
- [ ] Silence removal works.
